# Go HTTP Demo

Учебный проект для изучения основ клиент-серверного взаимодействия на языке Go.

Проект демонстрирует, как работает HTTP-сервер, как обрабатывать запросы и как клиент может взаимодействовать с сервером.

---

## Общая идея проекта

Проект состоит из двух частей:

### 1. Сервер (`server/`)
- обработка HTTP-запросов
- используется собственный маршрутизатор (`http.ServeMux`)
- реализованы несколько обработчиков (handlers)
- поддерживаются разные HTTP-методы (GET, POST) по главному маршруту ("/")
- возврат данных клиенту

### 2. Клиент (`client/`)
- отвечает за отправку HTTP-запросов
- проверка доступности сервера **Ping** (HEAD-запрос)
- выполнение GET-запроса с кастомными заголовками
- отправка POST-запроса с формой
- вывод полного ответа сервера (заголовки + тело)

---

## Как запустить:
1. В одном терминале: `cd server && go run .`
2. В другом терминале: `cd client && go run .`

### Сервер запустится на:
localhost:8080
