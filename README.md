Go HTTP Demo
Учебный проект для изучения основ взаимодействия **Client-Server** на языке Go.

## Функционал
* Проверка доступности сервера (**Ping**)
* Выполнение **GET** запросов с кастомными заголовками
* Отправка форм через **POST** запрос
* Обработка ошибок через **Sentinel Errors**

## Как запустить:
1. В одном терминале: `cd server && go run .`
2. В другом терминале: `cd client && go run .`
